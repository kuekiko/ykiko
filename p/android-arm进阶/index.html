arm-linux-gcc交叉编译器编译

arm-linux-gcc也能编译出ARM可执行文件。sudo apt install g++-arm-linux-gnueabihf

或者下载二进制文件安装。

arm-linux-gnueabihf-g++  -static  helloworld.cpp

push进Android之后运行成功
